# PRODIGY_GA_05

## Neural Style Transfer

This project applies the artistic style of one image to another image while preserving the original content.

## Technologies Used

- Python
- TensorFlow
- TensorFlow Hub
- Gradio
- Pillow

## Features

- Content Image Upload
- Style Image Upload
- Neural Style Transfer
- Interactive User Interface

## Workflow

Content Image
+
Style Image
↓
Neural Style Transfer Model
↓
Stylized Image

## Example

Content Image:
City Landscape

Style Image:
Van Gogh Painting

Output:
City image transformed with Van Gogh's artistic style.
